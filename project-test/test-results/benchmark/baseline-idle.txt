## To Capture Baseline

### 1. Memory (Idle)

```powershell
# Find server process
Get-Process -Name "*pcremote*" | Select-Object Id, ProcessName, @{N='Memory_MB';E={[math]::Round($_.WorkingSet64/1MB,2)}}

# Or using tasklist
tasklist /fi "imagename eq pcremote-server.exe" /fo csv
```

### 2. CPU (Idle)

```powershell
# Check CPU usage
Get-Process -Name "*pcremote*" | Select-Object Id, CPU, @{N='CPU_Pct';E={$_.CPU}}

# Or with typeperf
typeperf "\Process(pcremote*)\% Processor Time" -sc 5
```

### 3. Goroutines

```powershell
# Check pprof endpoint
curl http://localhost:6060/debug/pprof/goroutine?debug=1 | Select-Object -First 10
```
